# 监控助手 0.6.0

发布日期：2026-07-18

## 本次更新

- 新增“页面监控”模块，可直接配置单个页面的目标内容区域，自动发现网络 JSON 或静态 DOM 获取方案。
- 页面监控首次静默保存版本 1，后续保存历史版本、生成受限差异并复用 DeepSeek、企业微信和邮件通知。
- 公告详情首次基线失败后会自动补齐，避免只有第一页第一条公告存在历史版本而影响更新比对。
- 增加“采集配置可能失效”专项告警，与网络异常和登录失效分开判断、限频及恢复通知。
- 修复快速重定向页面被 `ERR_ABORTED (-3)` 误判为加载失败的问题。
- 修复连续创建任务时验证内容未清空的问题。
- 增加全局默认暂停抓取时段；任务可继承、覆盖或关闭。
- 修复安装包缺少 `nodemailer` 导致邮件发送失败的问题，并保留受限差异片段和当前版本全文。
- 全局日志标识触发任务；暂停状态颜色统一；全局通知配置标题布局修正。
- 菜单栏图标使用应用同风格模板图标；完整备份覆盖任务、历史、采集方案和全部设置。
- 侧边栏显示从应用包读取的实际版本号。

## 数据兼容性

- 兼容 0.5.x 已有公告任务和历史数据。
- 导入完整备份时同时支持公告任务和页面监控任务。
- 网站登录会话仍需在迁移后的设备重新登录，不包含在备份文件中。
